# MathMind Calculator

MathMind is a simple, visually appealing calculator web application. It features a modern design and provides basic arithmetic operations.

## Project Structure

The project consists of the following files:

- `index.html`: The main HTML file containing the structure of the calculator.
- `style.css`: The CSS file providing the styling for the calculator.
- `script.js`: The JavaScript file containing the logic for the calculator's functionality.

## Files Description

### index.html

The HTML file contains the following sections:

1. Header: Displays the title of the application.
2. Main Section: Contains the calculator display and buttons arranged in a grid layout.
3. Script: Links to the external JavaScript file.

### style.css

The CSS file styles the HTML elements to create a modern, user-friendly interface. Key styles include:

- Body: Sets the font and background color.
- Container: Centers the calculator on the page and adds padding, background color, and box shadow.
- Header: Styles the header with a background color and text alignment.
- Button Grid: Arranges the calculator buttons in a grid layout.
- Buttons: Styles the buttons with different background colors, border-radius, and hover effects.
- Display: Styles the calculator display for better readability.

### script.js

The JavaScript file contains the logic for the calculator's functionality. Key functionalities include:

- Display Update: Updates the display based on button clicks.
- Clear Button: Resets the calculator.
- Backspace Button: Deletes the last character.
- Arithmetic Operations: Performs basic arithmetic operations (addition, subtraction, multiplication, division).
- Calculate Function: Evaluates the expression and displays the result.

## How to Use

1. Clone the repository or download the files to your local machine.
2. Open `index.html` in a web browser to use the calculator.
