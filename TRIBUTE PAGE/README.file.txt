# Tribute to Albert Einstein

This project is a simple web page dedicated to the life and achievements of Albert Einstein. It includes sections for quotes, a brief biography, achievements, and inspiration drawn from Einstein's life and work.

## Files

- `index.html`: The main HTML file that structures the content of the tribute page.
- `styles.css`: The CSS file that styles the content of the tribute page.

## HTML Structure

The `index.html` file contains the following sections:

1. Header
    - Contains the main title of the page.

2. Hero Section
    - Displays an image of Albert Einstein.
    - Includes a blockquote with some of Einstein's famous quotes.

3. Biography Section
    - Provides a brief overview of Einstein's life and scientific contributions.

4. Achievements Section
    - Lists some of the major awards and honors Einstein received.

5. Inspiration Section
    - Highlights how Einstein's life and work have inspired others.

6. Footer
    - Contains a closing message and the author's signature.

## CSS Styling

The `styles.css` file includes styles for the following:

1. Body
    - Sets the default font and removes margin and padding.

2. Header
    - Styles the header with a background color, text color, and padding.

3. Hero Section
    - Applies a gradient background, centers the content, and styles the image and blockquote.

4. Biography, Achievements, and Inspiration Sections
    - Adds padding to these sections.
    - Styles the headings and paragraphs.

5. Achievements List
    - Removes default list styling and adds padding and borders to list items.

6. Footer
    - Styles the footer with a background color, text color, and padding.

## How to Use

1. Open `index.html` in a web browser to view the tribute page.
2. Modify the content in `index.html` to add more information or update existing content.
3. Adjust the styles in `styles.css` to change the appearance of the page.
