# Foodie Place

Foodie Place is a vibrant, user-friendly web page designed for a food ordering service. It features a variety of food items with ordering options and links to social media pages.

## Project Structure

The project consists of the following files:

- `index.html`: The main HTML file that contains the structure and content of the webpage.
- `foodie1.css`: The CSS file that styles the webpage.
- `https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css`: The external CSS file that provides icons from Font Awesome.

## Files Description

### index.html

The HTML file contains the following sections:

1. Header: Displays a marquee with a discount offer.
2. Title: Displays the title of the webpage.
3. Navigation: Contains links to different sections of the website.
4. Food Items Section: Displays various food items with images and order buttons.
5. Footer Navigation: Contains links to contact information and social media pages.

### foodie1.css

The CSS file styles the HTML elements to create a modern, user-friendly interface. Key styles include:

- Global Styles: Sets the font and removes default margin and padding.
- Header: Styles the marquee with a background color.
- Title: Centers the title and sets its color.
- Navigation: Styles the navigation bar with background color, text alignment, and hover effects.
- Food Items Section: Arranges the food items in a grid layout and styles the images and buttons.
- Responsive Design: Adjusts the grid layout for different screen sizes.

### External Font Awesome CSS

- Font Awesome: Adds icons for contact and social media links.

## How to View the Project

1. Clone the repository or download the files to your local machine.
2. Ensure you have an internet connection to load the Font Awesome icons.
3. Open `index.html` in a web browser to view the webpage.
